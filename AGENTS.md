## 📐 منهجية البناء

### قبل أي كود:
1. **Benchmark First** — ادرس ٣ منصات ناجحة قبل البناء
2. **Client-Side > Server-Side** كلما أمكن
3. **أقل اعتماديات خارجية** — صفر مكتبات لو مش ضرورية
4. **Copy Architecture — لا تنسخ كود**

### خلال البناء:
5. **عربي أولاً** — كل interface عربي RTL
6. **HTML Entities للعربي** — `&#XXXX;` = الحل الوحيد المضمون ضد تشوه الترميز
7. **لا تعديل على ملفات > 100KB** — استخدم Python script
8. **كل Fix = قابل للتراجع** — git checkout لو فشل

### بعد البناء:
9. **اختبر قبل ما تقول "تم"** — شوف النتيجة بعينك
10. **وثق الإصلاح** — كل مشكلة = درس في TOOLS.md
