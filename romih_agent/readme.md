# Romih Agent v1.0 🧠

**العقل المؤسسي الذكي — أول وكيل AI عربي متكامل.**

141 أداة | 6 بلجنز | Web UI | Telegram | محلي 100%

---

## 🚀 التشغيل السريع

### ويندوز:
```
دبل كليك على:  install.bat
أو:            run.bat
```

المتصفح يفتح تلقائياً على: `http://localhost:8800/romih/`

### من تيليجرام:
```
افتح @RomihAgentbot
أرسل /start
جرب: /goal أضف عميل جديد اسمه محمد
```

## 🧠 التقنيات

- **النموذج:** Google Gemini 2.5 Flash (OpenRouter)
- **Agent Loop:** Think → Plan → Execute → Observe
- **الوكلاء الفرعيون:** 5 Sub-agents
- **الأمان:** Safety Shield + Secrets Vault
- **الذاكرة:** 3 طبقات — قصيرة، جلسة، طويلة المدى
