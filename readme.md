# 🏥 مستشفى الشركات — Companies Hospital
> نظام تشغيل ذكي للشركات العربية الصغيرة والمتوسطة
> استقبال ← تشخيص ← علاج ← متابعة

---

## 🎯 الرؤية
"نظام التشغيل الأول للشركات العربية — حيث يلتقي الذكاء بالتنفيذ في منصة واحدة."

## 🏗️ التقنيات

| الطبقة | التقنية |
|---|---|
| واجهة | Next.js 14 + Tailwind + Shadcn/UI RTL |
| خادم | FastAPI Python 3.12 |
| بيانات | PostgreSQL 16 (Supabase) |
| كاش | Redis 7 (Upstash) |
| نشر | Railway + Vercel |

## 🚀 تشغيل محلي

```bash
docker-compose up -d     # PostgreSQL + Redis
cd backend && uvicorn app.main:app --reload
cd frontend && npm run dev
```
